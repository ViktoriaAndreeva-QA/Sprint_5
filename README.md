# Sprint_5: Автоматизация тестирования Stellar Burgers

## Описание проекта
Проект содержит автоматизированные тесты для сайта [Stellar Burgers](https://stellarburgers.education-services.ru/) с использованием Selenium WebDriver и pytest.


## Структура проекта
Sprint_5/
├── .gitignore                  # Исключаемые файлы
├── README.md                   # Документация
├── requirements.txt            # Зависимости
├── conftest.py                 # Фикстуры pytest
├── data.py                     # Генераторы тестовых данных
├── locators.py                 # Локаторы элементов
└── tests/                      # Тестовые файлы
    ├── test_registration.py        # Тесты регистрации
    ├── test_login.py               # Тесты входа
    ├── test_go_to_account.py       # Переход в ЛК
    ├── test_go_from_account.py     # Переход из ЛК
    ├── test_constructor.py         # Тесты конструктора
    └── test_logout.py              # Тест выхода


## Тестируемая функциональность

### 1. Регистрация (2 теста)
- Успешная регистрация с валидными данными
- Ошибка при пароле менее 6 символов

### 2. Вход в систему (4 теста)
- Вход по кнопке «Войти в аккаунт» на главной
- Вход через кнопку «Личный кабинет»
- Вход через кнопку в форме регистрации
- Вход через кнопку в форме восстановления пароля

### 3. Переход в личный кабинет (1 тест)
- Переход в личный кабинет

### 4. Переход из личного кабинета (2 теста)
- Переход из личного кабинета в конструктор
- Переход по логотипу Stellar Burgers

### 5. Конструктор (3 теста)
- Переход к разделу «Булки»
- Переход к разделу «Соусы»
- Переход к разделу «Начинки»

### 6. Выход из аккаунта (1 тест)
- Выход по кнопке «Выйти» в личном кабинете

**Всего: 13 автономных тестов**

## Особенности реализации
### 1.Генерация тестовых данных
- Для регистрации генерируются уникальные email в формате: имя_фамилия_номер_когорты_3цифры@yandex.ru
- Функция генерации: generate_email() в файле data.py
- Генератор подключен к тестам через фикстуру test_user

### 2.Автономность тестов
- Каждый тест открывает и закрывает браузер независимо
- Используется фикстура driver в conftest.py

### 3.Локаторы
- Все локаторы вынесены в отдельный файл locators.py
